01 — Python · Scikit-learn · XGBoost

ML for Steel Materials

12-day structured ML curriculum applied to a steel composition dataset. Predicting yield strength from Carbon %, Manganese %, and Rolling Temperature using Linear Regression, Random Forests, XGBoost, PCA, and K-Means clustering. Includes cross-validation, GridSearchCV, and metallurgical interpretation of results.

Regression Classification Clustering Dimensionality Reduction Materials Engineering

02 — PyRolL · Python · Rolling Simulation

Elastic Roll Gap Simulation — PyRolL Plugin

Self-study project building custom PyRolL plugins for elastic roll deflection using the Hitchcock model and slab analysis with disk elements. Covers full thermomechanical analysis of an IMF finishing sequence (F1–F4) for C45 steel, including force, strain, and temperature prediction. Rigid vs elastic gap comparison with Newton-Raphson neutral point solver.
